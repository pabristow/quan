Detailed Description

Uncertain class definitions.

Class for simple Propagation of Uncertainties according to a pure Gaussian model. Uncertain subroutines, manipulators & applicators. but not <iomanip> because defined here (as a copy of code in Std iomanip).

void setUncDefaults ( std::ios_base &  os)

Random id value.

Set 16 default values for uncertain output flags. Might be best to make setUncDefaults also set ios_base defaults with void setiosDefaults(ostream&);
Usage: 

 setUncDefaults(cout);
Warning:
setUncDefaults() MUST be called before using a stream.
 Default iword(*) values are all zero - these are not usable.
 Function \c setUncDefaults() sets some defaults
 Can set individual value, for example, for 3 sig Digits, os.iword(sigDigitsIndex) = 3;
 \note Index values must have been initialised by xalloc calls.
