Preface

This Quan library provides a collection of classes and functions for handling uncertain quantities - real values and measurements.

Uncertainties stored as notional standard deviation and degrees of freedom are propagated through calculations using the uncertain class type. Uncertainties are input and output including the uncertainty estimates typically from standard deviation and degrees of freedom, usually the number of observations less one.

See Quan uncertain classes library documentation.
